DriveAF

View image
DriveAF (Drive Ayman Farsi) is a Rust application that mounts cloud storages like Google Drive and Microsoft OneDrive on the filesystem. It can be used to add, edit, and delete cloud storages. Support for other cloud providers will be added after some testing. Currently, it is tested and supported on Windows and Linux. macOS support is planned.

Aeon

View image
Aeon is a one-place-for-everything kinda app. It is built using Rust with immediate GUI library egui. It is used to bookmark YouTube channels and watch their videos using mpv (mpvnet for windows). It also has built-in support for custom "windows" where draggable windows can be created to fetch and display information such as the economic calendar and TV shows release schedule.
